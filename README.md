# OnlineFooodDeliverySystem

Built a fullstack food delivery platform with AIpowered food comparison and integrated food vlogger post feed.Implemented a tokenbased feedback system where users earn tokens for reviews, usable for accessing AI comparison features.Developed an inapp social section for food vloggers, allowing users to view posts, comment, and discover tagged dishesrestaurants.Integrated a trained AI model to compare dishes based on specified Category.Used Spring Boot, React, MySQL to deliver a responsive and intelligent application for modern food users.
